Writes audio to a wav file.

The writer initializes the data_size to be 0. This will not be overwritten with the proper size until update_data_size is called.

While WavReader only requires its type be Seek to use the restart method, the WavWriter reqiures Seek for all types, because the final data size must be written to the header when the writer leaves scope.
